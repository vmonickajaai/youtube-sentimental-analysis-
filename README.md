# 🎥 YouTube Sentiment Analysis

A Machine Learning and Natural Language Processing (NLP) project that analyzes YouTube comments and classifies them as **Positive**, **Negative**, or **Neutral**.

---

## 📌 Project Overview

This project automatically determines the sentiment expressed in YouTube comments using NLP and Machine Learning techniques. It helps understand audience opinions, feedback, and engagement on YouTube videos.

---

## ✨ Features

- Analyze YouTube comments for sentiment
- Predict Positive, Negative, or Neutral sentiment
- Text preprocessing using NLP techniques
- Machine Learning-based sentiment classification
- Simple and interactive web interface
- Fast and accurate predictions

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/vmonickajaai/youtube-sentimental-analysis-.git
```

### 2. Move into the project

```bash
cd youtube-sentimental-analysis-
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python app.py
```

### 5. Open in your browser

```
http://127.0.0.1:5000
```

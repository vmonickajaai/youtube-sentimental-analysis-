🎥 YouTube Sentiment Analysis

A Machine Learning and Natural Language Processing (NLP) project that analyzes the sentiment of YouTube comments and classifies them as Positive, Negative, or Neutral. The project demonstrates an end-to-end sentiment analysis pipeline, including data preprocessing, feature extraction, model training, and prediction through a user-friendly interface.

📌 Project Overview

The objective of this project is to automatically determine the sentiment expressed in YouTube comments. By applying NLP techniques and machine learning algorithms, the system helps understand audience opinions, feedback, and engagement on YouTube videos.

🚀 Features
Analyze YouTube comments for sentiment.
Predict Positive, Negative, or Neutral sentiment.
Text preprocessing using NLP techniques.
Machine Learning model for sentiment classification.
Simple and interactive user interface.
Fast and accurate predictions.

⚙️ Installation
Clone the repository.
git clone https://github.com/vmonickajaai/youtube-sentimental-analysis-.git
Navigate to the project folder.
cd youtube-sentimental-analysis-
Install the required packages.
pip install -r requirements.txt
Run the application.
python app.py
Open your browser and visit:
http://127.0.0.1:5000
📊 Workflow
Collect YouTube comments.
Clean and preprocess the text.
Convert text into numerical features.
Train the sentiment analysis model.
Predict the sentiment of new comments.
Display the prediction to the user.
